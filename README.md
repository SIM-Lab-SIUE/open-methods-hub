# MC Research Methods: Open Methods Hub

[![License: CC BY 4.0](https://img.shields.io/badge/License-CC_BY_4.0-lightgrey.svg)](https://creativecommons.org/licenses/by/4.0/)
[![Quarto Website](https://img.shields.io/badge/Built_with-Quarto-blue)](https://quarto.org)

Course support website for **MC-451 Research Methods in Mass Media** at Southern Illinois University Edwardsville. The site structures a semester-long content analysis project into five progressive phases, guiding students from media consumers to published researchers.

**Live site:** <https://sim-lab-siue.github.io/liaison-program/>

**Instructor:** Dr. Alex P. Leith (aleith@siue.edu)

---

## Site Structure

| Page | Description |
|------|-------------|
| **Home** | Welcome page with course overview and quick-start links |
| **Get Started** | 15-minute workspace setup guide (Obsidian, Git, R, RStudio) |
| **Syllabus** | Full course syllabus with 17-week schedule and grading |
| **Phases** | Five progressive modules (see below) |
| **Resources** | Software installation guides, downloadable vault, templates |

### The Five Phases

| Phase | Role | Focus | Weeks | Points |
|-------|------|-------|-------|--------|
| I | The Journalist | Foundations — reading, reflecting, tools | 1-4 | 150 |
| II | The Architect | Design — theory, prospectus, archive research | 2-4 | 125 |
| III | The Builder | Methods — ethics, codebook, sampling, pilot | 5-12 | 250 |
| IV | The Analyst | Analysis — R programming, visualization, inference | 11-13 | 225 |
| V | The Publisher | Publishing — final portfolio, GitHub Pages | 15-17 | 250 |

---

## What's Included

- **Installation guides** for R/RStudio, VS Code, Git/GitHub, and Obsidian
- **Methods Vault** — a downloadable Obsidian workspace with templates for journals, literature notes, codebooks, and project deliverables
- **Weekly assignment templates** aligned with the 22-chapter textbook
- **Syllabus** with the full Fall 2026 schedule

## Building Locally

### Prerequisites

- [Quarto](https://quarto.org/docs/get-started/) (v1.3 or later)

### Render the site

```bash
quarto render
```

Output goes to `docs/`. Open `docs/index.html` in your browser.
